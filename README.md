# Reco — AI Career Intelligence

Reco is a Streamlit demo app that helps job seekers understand which roles to apply for, why they fit, what skills to improve, and how to tailor their resume.

## Features

- AI-style job match scoring
- Skill gap analysis
- Resume bullet improvement
- Salary intelligence
- Clean startup-style dashboard UI

## How to run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## How to deploy

Deploy on Streamlit Cloud:

1. Upload `app.py`, `requirements.txt`, and `README.md` to GitHub.
2. Go to Streamlit Cloud.
3. Create a new app.
4. Select this repository.
5. Set main file path as `app.py`.
6. Deploy.
